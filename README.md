# AXIOM Core — Invariant Semantic Embeddings

AXIOM is a small neural layer that learns invariant semantic fingerprints on top of existing embeddings.

It preserves meaning under:
- paraphrasing
- word reordering
- typos
- light prompt injection
- noise

Embedding → Invariant Gate → Semantic Fingerprint

AXIOM does NOT replace your embedding model.  
It wraps it.
